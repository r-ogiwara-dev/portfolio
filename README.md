# r.ogiwara Portfolio

🔗 https://portfolio-ten-ashen-b18ggrgiyh.vercel.app

## 技術構成

| 項目 | 技術 |
|---|---|
| フレームワーク | Next.js 16 (App Router) |
| 言語 | TypeScript |
| スタイリング | Tailwind CSS v4 + CSS Variables |
| アニメーション | CSS Keyframes / Intersection Observer API |
| イラスト | オリジナル SVG（インライン） |
| デプロイ | Vercel |
| バージョン管理 | GitHub |
| 開発環境 | WSL2 (Ubuntu) |

## ローカル起動

```bash
npm install
npm run dev
```
